# Java Console Calculator

## Objective
A basic console-based calculator built using Java.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Loop-based menu
- Divide-by-zero handling

## Concepts Used
- Methods
- Conditionals
- Loops
- Scanner (I/O)
- OOP basics

## How to Run
1. Compile:
   javac Calculator.java
2. Run:
   java Calculator
